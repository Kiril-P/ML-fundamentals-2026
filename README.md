# ML Fundamentals 2026

Repository for Individual Assignment I in Machine Learning Foundations.

## Repository Link
https://github.com/Kiril-P/ML-fundamentals-2026

## Project Structure
- `assignment_1_kiril_petrovski.ipynb`: main notebook for the assignment
- `data/raw/`: contains `bank-additional.csv`
- `data/processed/`: optional location for intermediate datasets if needed
- `figures/`: optional location for exported plots or confusion matrices

## Quick Start
1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Download the UCI Bank Marketing file `bank-additional.csv`.
4. Put it in `data/raw/`.
5. Open `assignment_1_kiril_petrovski.ipynb` and run the notebook from top to bottom.
